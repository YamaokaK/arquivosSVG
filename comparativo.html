PNG

Os PNG (Portable Network Graphics) são arquivos rasterizados, eles têm resolução alta e poder de processar milhões de cores, o que faz dele uma excelente escolha para várias situações, mas não são recomendados para fotos digitais por resultar em um arquivo muito grande.


SVG x PNG


Resolução

PNG é um formato de imagem rasterizado, ou seja, baseado em pixels. Por mais que processem altas resoluções, as imagens em PNG quando muito expandidas ou muito diminuídas, podem ficar irreconhecíveis. Por outro lado, os SVG são vetoriais, que significa que são criados através de uma rede temática complexa, o que a torna capaz de ser expandida para qualquer tamanho sem perder sua resolução.


Tamanho

Os arquivos PNG são grandes, o que os fazem capaz de processar ótimas resoluções, porém devido ao tamanho, levam mais tempo para serem processadas, o que pode fazer uma pagina levar mais tempo para ser carregada, já os SVGs são menores e não costumam deixar o computador ou site lento, além de ter a flexibilidade de tamanho sem perder qualidade.


Compactação

Os SVGs oferecem compactação sem perdas, ou seja, podem ser diminuídos sem perder definição, detalhe ou qualidade, no entanto, apesar de oferecer uma compactação sem perdas de 5% a 20%, eles provavelmente ainda serão maiores que os SVGs.


Uso na Web

Além de serem menores, os SVGs têm uma vantagem importante: a linguagem XML, o que significa que cada arquivo exibido está em texto, não código. Como resultado, os leitores de tela e mecanismos de pesquisa podem analisá-los, o que os torna ideais para acessibilidade e SEO.


Compatibilidade

Apesar de ser considerado mais avançado, os arquivos SVGs não são compatíveis com todos os navegadores e sistemas operacionais, além disso, a linguagem de programação e composição vetorial podem ser desconhecidas e difíceis para usuários novos. PNGs por outro lado são um formato online padrão, com suporte amplo de navegadores e sistemas operacionais.


Animação

Os arquivos SVG aceitam animações, mas não são facilmente acessíveis como outros formatos, já os PNG não são compatíveis com animações.



EPS

O EPS (Encapsulated PostScript) é o formato padrão do setor de impressão, é a principal escolha quando se trata de anúncios grandes como pôsteres e outdoors. É amplamente compatível com vários sistemas operacionais, programas de design e outros, mas muitas pessoas veem o EPS como um arquivo legado, sendo assim então gradualmente substituído por outros arquivos como PDF.


EPS x SVG


Uso na Web

Os arquivos EPS foram lançados em 1992, quando o cenário digital era bem diferente, por isso, esse formato nem tem boa otimização para web. Já os arquivos SVG usam dados de imagem e texto. A linguagem XML armazena informações de texto com texto de verdade, e não substitui por formas ou pontos, assim os mecanismos de pesquisa podem identificar o texto SVG.


Imprimir

Os arquivos EPS são uma ótima opção se você trabalha com uma gráfica que usa equipamentos mais antigos, já que é mais propensa a ter problemas com tipos de arquivos modernos. Você pode usar arquivos SVG em alguns tipos de impressões, principalmente de roupas e papelaria personalizada, mas eles não são compatíveis com dispositivos suficientes para uso em tarefas cotidianas.


Escalabilidade

Os SVGs são usados na Web, então não costumam ser ideais para impressão em grande escala. Eles são melhores para exibição na tela do computador. Sendo assim, quando se trata de gráficos, tabelas ou infográficos detalhados que precisam de zoom, o formato SVG é a escolha perfeita. Os arquivos EPS são ideais para impressão e costumam ser usados em outdoors e outros anúncios impressos grandes. Eles armazenam informações adicionais de programação sobre cor e tamanho para ajudar as impressoras a gerenciar a tarefa.


Tamanho

Por armazenar informações sobre a imagem, os arquivos EPS acabam sendo maiores, tornando os arquivos SVG uma melhor alternativa quando se trata de desempenho.
